Overview

Hisuri is a registration platform with both a website and a companion mobile app. During my internship I worked on it from design through to a working product, focusing on a smooth sign-up flow and a clean, responsive interface.

What I worked on

  • Built out the website and a cross-platform mobile app with React Native.
  • Translated designs into responsive, accessible interfaces.
  • Implemented the registration flow and connected it to the backend.
  • Made functional and technical improvements across the product.
  • Experienced the full development process — design, build, test, ship.

Tech

  • React Native
  • JavaScript
  • HTML
  • CSS

What I learned

Working on a real product with real users taught me how the pieces fit together — from turning a design into working screens, to shipping improvements as part of a team and seeing a feature go all the way to production.
